# Step 002: Initialize Next.js Project

> **MANUAL STEP** — requires human action.

## Description

Initialize a new Next.js project with TypeScript, App Router, and Tailwind CSS using the official scaffolding tool. This creates the foundational project structure that all subsequent steps build upon.

## Requirements

- Use `npx create-next-app@latest` to scaffold the project
- Select App Router (not Pages Router)
- Enable TypeScript
- Enable Tailwind CSS
- Enable ESLint
- Use `src/` directory structure
- Set import alias to `@/`
- Project name: `scout-bingo`

## Files to Create/Modify

- `package.json` — generated by create-next-app with Next.js, React, and TypeScript dependencies
- `tsconfig.json` — generated with TypeScript configuration and `@/` path alias
- `next.config.ts` — generated Next.js configuration
- `src/app/layout.tsx` — generated root layout
- `src/app/page.tsx` — generated home page
- `src/app/globals.css` — generated global styles with Tailwind directives
- `tailwind.config.ts` — generated Tailwind configuration
- `postcss.config.mjs` — generated PostCSS configuration
- `.eslintrc.json` — generated ESLint configuration
- `.gitignore` — generated gitignore
